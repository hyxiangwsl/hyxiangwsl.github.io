从0开始搭建ssr服务器(DigitalOcean)

2020-02-02 — 作者: yuxiang — 1 分钟阅读

1.购买服务器

使用github学生包

学生认证可得到DO服务器50刀,大概能用一年，充值5刀进去，就能享受50刀 传送门 前提是要在github上先通过学生认证，充值5刀进你的DO账户，这一步请百度

如果打不开网站，则需要在C:\Windows\System32\drivers\etc文件目录下的hosts文件下添加一行，用记事本或者notepad++打开

199.27.79.249                cloud-cdn-digitalocean-com.global.ssl.fastly.net

购买服务器

选择自己熟悉的操作系统centos 选择最低配置5$每月 选择机房ip地址，可以先测速 测速地址 远程连接，通过ssh连接，添加一个ssh密钥，通过xshell的工具向导新建一个ssh密钥，把本地生成的密钥复制到框里就行 参考做法 这里填你的主机名字和标签 然后点击最底下的create droplets就可以了

成功！

2.使用xshell远程连接服务器

软件传送门 密码：lzuz 直接点击安装就可以了 刚开始连接会有是否接受密钥的弹出框，一致性接受就可以了，填写登录用户root,一个用户和密码,选择你在DO上添加的密钥，密码是写密钥的密码（用xshell新建密钥会叫你填写密码） 然后就可以连接成功。

3.搭建ssr加速

接下来就是安装 SS和SSR了

这里用的是逗比的兼容的脚本，如果没有 wget的话 先输入yum install -y wget

wget -N --no-check-certificate https://raw.githubusercontent.com/ToyoDAdoubiBackup/doubi/master/ssrmu.sh && chmod +x ssrmu.sh && bash ssrmu.sh

然后输入 1 回车安装 IP的话 回车就行 用户名自己取一个，然后端口也是随便一个 在 1~65535范围内就行，密码的话自己填，最后也复杂一点 加密我选的是aes-128-ctr协议,插件和混淆插件我都是选的第一个，限制连接的设备数自己选，限速也是，限制用的流量也自己填，然后一路回车等待安装完成 若出现上面的管理界面，输入数字1来安装SSR服务端。 使用sh ssr.sh快捷管理命令，再尝试

接着就是打开SS或SSR复制连接进去

软件传送门 提取码：km08 也可以填写你服务器配置的信息

下面是BBRPlus的安装

(把下面的全部复制到xshell) 如果用的是Centos 必须使用7.x的版本 不支持6.x的版本

 wget -N --no-check-certificate "https://raw.githubusercontent.com/chiakge/Linux-NetSpeed/master/tcp.sh" && chmod +x tcp.sh && ./tcp.sh

选择 2 安装bbrplush内核 然后等待安装完成，然后重启

重启完成后 重新连接，如果你用的不是root用户 先 sudo -i 输入回车

然后输入 ./tcp.sh 回车 选择 7 加速完成
